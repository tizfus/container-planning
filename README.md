# Container Planning

This project wants to provide a tool for planning the filling of a shipping container.

## Project structure

The tool is a Single-page application made in plain JavaScript/HTML wihout using any framework to generate the content; However, "agentic development" will be used extensively, as this project is also intended to be an exercise for this type of development.
All the source code is located in the `/src` directory.

All tests will be saved in the `/tests` directory and are written in Gherkin. From the root directory, you can run them using `npm --prefix ./tests/ `
